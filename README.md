# Amazon Homepage Clone

A responsive Amazon Homepage Clone built using **HTML5** and **CSS3**. This project recreates the look and feel of the Amazon landing page while focusing on front-end layout design and responsive styling.

## 🚀 Features

- Responsive Navigation Bar
- Search Bar
- Hero Banner
- Product Cards
- Multi-section Footer
- CSS Flexbox Layout
- CSS Grid Layout
- Hover Effects
- Font Awesome Icons
- Clean and Reusable UI Components

## 🛠️ Tech Stack

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Font Awesome

- Built responsive layouts
- Implemented reusable UI components
- Enhanced understanding of modern CSS styling
